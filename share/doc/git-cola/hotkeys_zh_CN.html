键盘快捷键
    提交
Ctrl + Return : 提交已缓存变更
Ctrl + s : 缓存所选项目
Alt + a : 缓存已更改项目
Alt + u : 缓存未追踪项目
Ctrl + u : 还原缓存改动
Ctrl + z : 撤销未缓存变更
Shift + Ctrl + m : Merge branches
Ctrl + m : 修正前一次提交
Shift + Ctrl + c : Cherry pick
    操作
Ctrl + b : 建立新分支
Alt + b : 取出分支內容
Ctrl + t, t : Find files
Ctrl + g : 搜索
Alt + d : 显示内容差异
Alt + e : 导出差异
Shift + Alt + s : 缓存
Ctrl + backspace : Delete untracked files
Shift + Ctrl + f : Toggle paths filter
Ctrl + r : 重新整理
? : 显示键盘快捷键列表
    编辑
Ctrl + e, Enter : 启动文本编辑器
空白鍵 : 用默认程序打开
Shift + 空白鍵 : 打开上级目录
Shift + Ctrl + e : 查看 / 编辑最近修改的文件
Ctrl + d : 使用「git difftool」命令查看内容差异
Ctrl + # : 聚焦于工具
Shift + Ctrl + # : 显示/隐藏工具
    差异查看器
s : 缓存所选项目，未选择项目时与h作用相同
    「浏览器」窗口操作
Shift + Ctrl + h : 查看变更历史
Shift + Ctrl + d : 与之前版本比较差异
    导航键
h : 左移／折叠
j : 下移
k : 上移
l : 右移／展开
Ctrl + l : 聚焦在 DAG 输入框
